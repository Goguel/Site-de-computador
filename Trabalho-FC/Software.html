Conceito de software

    O software é a parte lógica de um computador. Uma sequência de informações a serem seguidas ou executadas na manipulação, redirecionamento ou modificação de dados/informações. Os softwares podem ser classificados em relação a finalidade ou as regras de utilização.

Classificação de acordo com :

    Finalidade

  • Software Básico ou Software de Sistema: realiza funções vitais para o bom uso do computador. Exemplo: Sistemas operacionais como Windowns, IOS, Linux etc.

  • Software de Aplicativo: aplicações criadas para solucionar problemas específicos. Exemplos: Word, power point, google chrome etc.

    • Regras de utilização

  • Software propietário

    • * Não disponibiliza o código fonte
    * Não é permitido redistribuição
    * Possui um custo de aquisição
    * Exemplo:Windows

  • Software livre

    • * Disponibiliza o código fonte
    * É permitido redistribuição
    * Software livre é uma questão de liberdade, não de preço
    * Exemplo: Linux
